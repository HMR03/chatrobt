# AI Chat — 桌面 AI 聊天机器人

基于 **Tauri 2.x + React 19 + TypeScript + Tailwind CSS v3** 构建的桌面端 AI 聊天应用，接入 DeepSeek V4 API，支持多人格、技能系统、记忆系统和 SQLite 持久化存储。

## 三、核心功能

### 3.1 DeepSeek API 接入

- **端点**：`https://api.deepseek.com/chat/completions`（OpenAI 兼容格式）
- **模型**：`deepseek-v4-flash`（快速）/ `deepseek-v4-pro`（高质量）
- **流式输出**：SSE 协议，`AsyncGenerator` 逐字产出，前端打字机效果渲染
- **错误处理**：网络异常 + API 错误码统一捕获

### 3.2 多人格系统

| 人格 ID | 名称 | Temperature | 特长 |
|---------|------|-------------|------|
| `assistant` | 🤖 通用助手 | 0.7 | 准确、有条理、双语交流 |
| `writer` | ✍️ 创意写作 | 1.2 | 文学性强、多风格创作 |
| `coder` | 💻 编程导师 | 0.3 | 循序渐进、中英混合教学 |

- 每个人格有独立的 system prompt / greeting / model / temperature / maxTokens / skills
- 配置存储在 `localStorage`，可扩展至 `appDataDir/personas/` JSON 文件
- 内置人格不可删除

### 3.3 技能系统（Skill）

| 技能 ID | 名称 | 类型 | 说明 |
|---------|------|------|------|
| `current-time` | 当前时间 | `function_call` | DeepSeek Tool Use，执行器返回本地时间 |
| `translator` | 翻译助手 | `prompt_injection` | 追加翻译规则到 system prompt |
| `summarizer` | 网页摘要 | `prompt_injection` | 追加摘要规则到 system prompt |

- `function_call` 型：定义 JSON Schema → 模型调用 → 本地执行 → 结果回传
- `prompt_injection` 型：动态追加到 system prompt 末尾
- 每个技能可在人格编辑面板独立开关

### 3.4 记忆系统

- 每 10 轮对话触发一次提取
- 调用 DeepSeek API（temperature 0.1）提取事实/偏好/话题
- 去重后存入 SQLite，下次对话自动注入 system prompt
- 记忆限 50 条/人格，按时间倒序

### 3.5 数据持久化

- **SQLite**：会话表 + 消息表 + 记忆表（三级联删）
- **安全存储**：API Key 通过 `tauri-plugin-store` 加密存储
- **浏览器回退**：在 `npm run dev` 模式下自动使用内存 Mock

### 3.6 系统集成

- **系统托盘**：关闭窗口隐藏到托盘，托盘菜单"显示窗口"/"退出"
- **全局快捷键**：`Alt+Space` 切换窗口显示/隐藏
- **开机自启**：`tauri-plugin-autostart`（macOS LaunchAgent）
- **安全策略**：CSP 限制只允许连接 `api.deepseek.com`

## 五、数据流

```
用户输入
    │
    ▼
useChat.sendMessage()
    │
    ├─ 构建 system prompt (persona + skills + memories)
    ├─ 收集 function_call 型 tool schemas
    │
    ▼
streamChatCompletion()  ←── DeepSeek API (SSE)
    │
    ├─ type: "content"    → 逐字更新 UI
    ├─ type: "tool_calls" → 执行本地函数 → 回传结果 → 再次流式调用
    ├─ type: "done"       → 结束，触发记忆提取
    └─ type: "error"      → 显示错误
    │
    ▼
AppContext 全局状态同步
    ├─ localStorage (配置/人格/技能)
    └─ SQLite (会话/消息/记忆)
```

## 六、运行指南

### 环境要求

- **Node.js** ≥ 18
- **Rust** (rustup) + MSVC 工具链
- **WebView2**（Windows 10+ 自带）

### 命令

```bash
# 安装依赖
npm install

# 开发模式（热更新）
npm run tauri dev

# 生产构建
npm run tauri build

# 仅类型检查
npx tsc --noEmit
```

### 首次使用

1. 启动应用后点击左下角 **设置**
2. 输入 DeepSeek API Key（格式：`sk-...`）
3. 选择默认人格开始对话
4. 按 `Alt+Space` 可随时呼出/隐藏窗口
